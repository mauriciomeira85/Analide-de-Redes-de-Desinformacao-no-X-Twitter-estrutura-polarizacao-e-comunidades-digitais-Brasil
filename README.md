# Analide-de-Redes-de-Desinformacao-no-X-Twitter-estrutura-polarizacao-e-comunidades-digitais-Brasil
Este repositório apresenta um estudo técnico em desenvolvimento sobre a estrutura das redes de circulação de conteúdo político no X (Twitter). O projeto busca identificar padrões de polarização, centralidade e formação de comunidades digitais, contribuindo para o debate sobre desinformação e integridade informacional no Brasil.
